Python

Data Science is in charge of analyzing, transforming data and extracting useful information for decision making. Thanks to Python, these tasks can be performed without advanced programming skills, with few lines of code and in friendly programming environments that facilitate programming and visualization of results.

Python is an interpreted, multi-paradigm, cross-platform programming language, mainly used in Big Data, AI (Artificial Intelligence), Data Science, testing frameworks and web development. This makes it a high-level general purpose language due to its extensive library, whose collection offers a wide range of facilities. It has become one of the top choices for doing Data Science due to its open source nature, its simplicity, the support of the developer community and its extensive library.
